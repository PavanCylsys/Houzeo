# Houzeo Real Estate UI (Vue)

Responsive real estate listing interface inspired by Houzeo, built using Vue 3 and Tailwind CSS.

## Features
- Responsive layout (Desktop & iPhone 14 Pro Max)
- Property cards with hover effects
- Image slider with lazy loading
- Pulse animation on favorite iconI
- Map + listings layout
- Clean, modular Vue components

## Tech Stack
- Vue 3 (Composition API)
- Tailwind CSS
- Vite

## Setup Instructions

```bash
npm install
npm run dev
